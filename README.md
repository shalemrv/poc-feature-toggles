# Project and Invoice Creation Backend

Postman collection can be found in the codebase.

> Add an environment variable in Postman
> 
> `BASE` = `http://127.0.0.1:8000`

| Entity        | Operations    |
| ------------- | ------------- |
| Projects      | CRUD          |
| Invoices      | -R--          |
| FeatureFlags  | CRUD          |
________
### Setup Instructions

```console
  composer install
```

For Windows
```console
  init.bat
```

For Linux
```console
  bash init.bat
```
